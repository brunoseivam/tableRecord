=title Table Record (table)

The table record stores a multi-column table using per-column field bundles
(analogous to the aSub record's A..U input fields). Each column has its own
NAME, LABEL, TYPE, INP, VAL, and CHGD fields. qsrv publishes this record as
an NTTable.

=recordtype table

=cut
