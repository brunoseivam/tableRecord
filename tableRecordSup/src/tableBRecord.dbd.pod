=title Table Record Design B (tableB)

The tableB record stores a multi-column table using per-column field bundles
(analogous to the aSub record's A..U input fields). Each column has its own
NAME, FTVL, INP, NELM, NORD, VAL, and CHGD fields. qsrv publishes this
record as an NTTable.

=recordtype tableB

=cut
